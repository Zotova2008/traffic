# FitHub Tariffs - Тестовое задание Frontend

## Описание проекта

Веб-приложение для отображения тарифов фитнес-платформы с интерактивным таймером, анимациями и функционалом выбора
тарифа.

## Технологический стек

- **React** 18.3.1
- **TypeScript** 5.6.2
- **Next.js** 13.x
- **Tailwind CSS** 3.4.19
- **shadcn/ui** - UI компоненты

## Реализованный функционал

### 1. Таймер в закреплённом хедере

- Обратный отсчёт с 2 минут
- При остатке 30 секунд таймер мигает и выделяется красным цветом
- При истечении времени показывается анимированное уведомление

### 2. Карточки тарифов

- Данные загружаются с API: `https://t-core.fit-hub.pro/Test/GetTariffs`
- Процент скидки рассчитывается автоматически
- Тариф с `is_best: true` отображается увеличенной карточкой
- При выборе карточка подсвечивается оранжевой рамкой
- При истечении таймера скидочные цены плавно исчезают, отображаются полные цены

### 3. Анимация кнопки "Купить"

- Эффект мигания (pulsing glow) без перехода
- Реализован через CSS keyframes animation

### 4. Валидация чекбокса

- При нажатии на кнопку без проставленного чекбокса:
    - Чекбокс подсвечивается красным
    - Текст становится красным
    - Добавляется анимация shake

### 5. Анимация истечения таймера

Когда таймер заканчивается:

- Появляется модальное окно с анимацией "Время истекло!"
- Скидочные бейджи плавно исчезают (slide up + fade out)
- Цены меняются местами с анимацией pulse
- Зачёркнутая цена меняется на старую скидочную цену
